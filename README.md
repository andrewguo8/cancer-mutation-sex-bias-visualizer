# cancer-mutation-sex-bias-visualizer

An iOS application built to detect, analyze, and visualize sex-based biases in cancer mutations using open-source clinical oncology data. 

## The Problem & Insight
Clinical treatments often overlook how biological sex impacts cancer mutation rates. This app parses public genomic databases to visually highlight statistically significant sex biases in specific cancer types (like LGG/Lower Grade Glioma), helping researchers identify targets for personalized, sex-specific therapies.

## Tech Stack
- **Language:** Swift
- **Framework:** SwiftUI / Xcode
- **Data Source:** cBioPortal

## Features
- **Dynamic Filtering:** Sort mutation frequencies by cancer type, gene, and study.
- **Data Visualization:** Graphs and data tables for each mutation.
